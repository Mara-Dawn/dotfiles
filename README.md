# My Dotfiles
use `stow . --adopt` in dotfiles dir to apply config to home folder.

# Dependencies
* zsh
* hyprland
* hyprpaper
* hyprlock
* hyprpanel
* kitty
* starship
* rofi

# Optional Tools
* btop
* cava
* clipse
* direnv
* lazygit
* neovim
* thunderbird
* tmux
* vesktop
* yazi
